# Geometry

## Testing using Mocha and Chai

- [Mocha](https://mochajs.org/)
- [Chai BDD](http://chaijs.com/api/bdd/)

Install the latest official version of Mocha

```bash
$ npm install -g mocha
```

Install dependencies

```bash
$ npm install mocha chai --save-dev
```

Write tests

- 'test/*.js'

Run tests

```bash
mocha
```

or

```bash
npm run test
```
